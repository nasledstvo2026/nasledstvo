Социальный консультант

Мультиагентная система консультирования по мерам социальной поддержки

Социальный консультант — интеллектуальная система на базе двух специализированных агентов (социальный-агент + социальный-верификатор), которая обеспечивает консультирование по трём ключевым категориям социальной поддержки: инвалидность (181-ФЗ), ветераны боевых действий (5-ФЗ), участники СВО и члены их семей.

Архитектура

агент-консультант социальный-агент — ведущий юрист-эксперт по соцподдержке. Отвечает на вопросы пользователей, формирует еженедельные обзоры, ведёт базу знаний.
агент-верификатор социальный-верификатор — судебный архивный эксперт. Проверяет юридические факты по официальным источникам (гарант → cntd → консультант → СФР → ФНС).
skill social-verifier-protocol — единый протокол верификации: формат запроса, приоритет источников, система статусов (✅/❌/🔄/❓).

Три специализации

👥 Люди с инвалидностью
Базовый документ: 181-ФЗ «О социальной защите инвалидов». ЕДВ, НСУ, пенсии, льготы ЖКУ, доступная среда.
🎖️ Ветераны боевых действий
Базовый документ: 5-ФЗ «О ветеранах» (ст. 3, 16). ЕДВ — 3 896 руб./мес, льготы ЖКУ — 50%, транспорт.
⚔️ Участники СВО и члены семей
Указы Президента № 98, № 100. Выплата при ранении — 3 млн руб., ежемесячная компенсация семье погибшего — 28 000 руб.
