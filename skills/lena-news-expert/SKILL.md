4. **MOEX ISS API** — данные по облигациям РЖД 1Р-37R

## База знаний новостей (knowledge/lena/news-knowledge.md)
Что должна содержать:
1. Категории новостей: законы (ФЗ, ПП, проекты), судебная практика, банки, нотариат, СФР, налоги, недвижимость, СВО
2. Шаблоны для каждого типа новости: заголовок, суть, для граждан, источник, тэги
3. Список проверенных источников и их тэгов (rg.ru, pravo.gov.ru, notariat.ru, banki.ru, duma.gov.ru)
4. Ключевые темы-маяки: Цифровое наследство, ЕИС нотариата, жилищные сбережения, сальдо ЕНС
5. Дедупликация: алгоритм проверки по URL + заголовку + дате + источнику
6. Формат дайджеста: Главное (1-2 новости с разбором), Коротко (3-5 новостей сводкой)

## База знаний РЖД (knowledge/lena/rzd-knowledge.md)
Что должна содержать:
1. ISIN: RU000A10AZ45
2. Параметры выпуска: номинал, купон, даты, оферта, амортизация
3. Ключевые индикаторы: цена, доходность, спред к ОФЗ, дюрация
4. Формат итогов торгов: цена закрытия, средневзвешенная, изменение, доходность, объём, сделки
5. Нормальные диапазоны для этого выпуска
6. Источники данных: MOEX ISS API (get_board_stateboard), rusbonds.ru
