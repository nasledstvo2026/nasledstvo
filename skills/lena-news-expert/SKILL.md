Специалист по мониторингу новостей и законодательства по наследству + облигации РЖД 1Р-37R. Для Лены (@lena, ID: 254785028).

## Файлы навыка
- `memory/lena-news-daily.md` — дневной дайджест новостей
- `memory/lena-news-seen.md` — дедупликация (история URL/заголовков)
- `memory/lena-rzd-daily.md` — итоги торгов РЖД 1Р-37R

## Источники
1. **LegalMCP** (legalmcp.ru) — для законодательных изменений по наследству (ГК РФ, ФЗ о нотариате, НК РФ)
2. **web_search** — поиск новостей, законодательных проектов
3. **Домены для search-agent:** tass.ru, rbc.ru, kommersant.ru, vedomosti.ru, rg.ru, pravo.gov.ru, notariat.ru, banki.ru, duma.gov.ru, consultant.ru, garant.ru, nalog.gov.ru, sfr.gov.ru, mintrud.gov.ru
4. **MOEX ISS API** — данные по облигациям РЖД 1Р-37R (rzd-search-agent → rzd-raw.json)
5. **rusbonds.ru / cbonds.ru** — справочные данные по бондам (fallback для search-agent)

## База знаний новостей (knowledge/lena/news-knowledge.md)
Что должна содержать:
1. Категории новостей: законы (ФЗ, ПП, проекты), судебная практика, банки, нотариат, СФР, налоги, недвижимость, СВО
2. Шаблоны для каждого типа новости: заголовок, суть, для граждан, источник, тэги
3. Список проверенных источников и их тэгов (rg.ru, pravo.gov.ru, notariat.ru, banki.ru, duma.gov.ru)
4. Ключевые темы-маяки: Цифровое наследство, ЕИС нотариата, жилищные сбережения, сальдо ЕНС
5. Дедупликация: алгоритм проверки по URL + заголовку + дате + источнику
6. Формат дайджеста: Главное (1-2 новости с разбором), Коротко (3-5 новостей сводкой)

## База знаний РЖД (knowledge/lena/rzd-knowledge.md)
Что должна содержать:
1. ISIN: RU000A10AZ45
2. Параметры выпуска: номинал, купон, даты, оферта, амортизация
3. Ключевые индикаторы: цена, доходность, спред к ОФЗ, дюрация
4. Формат итогов торгов: цена закрытия, средневзвешенная, изменение, доходность, объём, сделки
5. Нормальные диапазоны для этого выпуска
6. Источники данных: MOEX ISS API (get_board_stateboard), rusbonds.ru

## Принципы
1. Лена — консультант для граждан. Язык простой, понятный.
2. Каждая новость — с пометкой "для граждан" (что это значит для обычного человека)
3. Дедупликация ОБЯЗАТЕЛЬНА: перед добавлением проверять lena-news-seen.md

9. Лимиты: search ≤15 новостей, verify ≤10, html ≤7 (2+5)
