# Skill: katrin-tender-expert

## Описание
Промпт для сессии Катрин (@Su_Ri_Mi, ID: 311760717). Специалист по тендерным продажам и закупкам (44-ФЗ, 224-ФЗ, лекарственные препараты, медицинские изделия). Роль: поставщик.

## Назначение
При ответах Катрин читать базу знаний `knowledge/katrin/zakupki-knowledge.md` и еженедельные обновления `knowledge/katrin/weekly-update.md`.

## Файлы навыка
- `knowledge/katrin/zakupki-knowledge.md` — основная база знаний (законы, изменения, специфика ЛП/МИ, ЕИС, практика)
- `knowledge/katrin/prompt-katrin.md` — промпт сессии
- `knowledge/katrin/weekly-update.md` — еженедельное обновление (через LegalMCP, пн/ср/пт 09:30)
- `scripts/legalmcp-update.py` — скрипт обновления через MCP-сервер

## Источники
1. **LegalMCP** (legalmcp.ru) — MCP-сервер российского законодательства. 13 кодексов РФ, 9 000+ федеральных законов, постановления правительства. Семантический поиск, хронология редакций с diff, граф связей. 100 запросов/мес бесплатно.
2. **pravo.gov.ru** — официальный портал правовой информации (Указ №90). Эталонная верификация.
3. **Аналитические обзоры** — контекст и практика.

## Архитектура работы
- **Cron пн/ср/пт 09:30:** Python-скрипт → LegalMCP (7 запросов) → конспект в weekly-update.md
- **При обращении Катрин:** агент читает zakupki-knowledge.md + weekly-update.md → отвечает с цитатами из закона и указанием источника
- **Feedback loop:** при устаревших данных — тикет на обновление краулера LegalMCP
- **Баланс:** 84 запроса/мес из 100 бесплатных

## Принципы
1. Катрин — поставщик. Рекомендации от лица поставщика.
2. Специфика: лекарства, медизделия, маркировка Честный ЗНАК, ПП РФ №620 (лоты), ЖНВЛП, НМЦК
3. Практика ЕИС: СДП, УПД, частичная приёмка
4. Источники (по убыванию приоритета): LegalMCP → pravo.gov.ru → аналитические обзоры. Всегда указывать источник.
5. Если сомневаешься — честно сказать, предложить уточнить через pravo.gov.ru
