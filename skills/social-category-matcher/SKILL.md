# social-category-matcher

## Описание
Скилл для категоризации вопросов пользователей по социальной поддержке в РФ. Определяет категорию (инвалидность / ВБД / СВО / не тема) и маппит на релевантные разделы базы знаний + соответствующие статьи НПА.

## Когда использовать
Когда social-agent получает вопрос от Розы или Ирины — **перед** подготовкой ответа.

## Процедура

### Шаг 1. Извлечь суть запроса
- Определи, о какой категории граждан вопрос:
  - **Инвалидность** (I, II, III группа, дети-инвалиды, инвалиды с детства, инвалиды боевых действий)
  - **Ветераны боевых действий (ВБД)** (по ст. 3 5-ФЗ, подпункты 1-9)
  - **СВО** (мобилизованные, контрактники, добровольцы, члены семей)
  - Если вопрос не относится ни к одной — вежливый отказ с указанием профиля
  - Если относится к нескольким — все категории

### Шаг 2. Загрузить релевантные разделы базы знаний
В зависимости от определённой категории — прочитать конкретные файлы:

| Категория | Файлы базы знаний |
|-----------|------------------|
| Инвалидность | `federal-law.md` (181-ФЗ, раздел про инвалидов) + `payments-table.md` (ЕДВ/НСУ) + `categories.md` (инвалиды) |
| ВБД | `federal-law.md` (5-ФЗ, ст. 3, ст. 16) + `payments-table.md` (ЕДВ ВБД) + `categories.md` (ВБД) |
| СВО | `svo-decrees.md` + `federal-law.md` (указы по СВО) + `payments-table.md` (участники СВО) + `categories.md` (СВО) |
| Если регион = Москва (по доп. вопросу) | `moscow-region.md` |

**Не читать** файлы, не относящиеся к категории — это лишние токены.

### Шаг 3. Сформировать факты для верификации
Для каждого утверждения из ответа (сумма, дата, статья НПА) — сформировать факт в формате:
```
Факт N: [утверждение] — [источник: файл/статья/URL]
```

### Шаг 4. Передать верификатору
Вызвать `sessions_send(agentId="social-verifier", message="...")` со всеми фактами.

### Шаг 5. Только после вердикта — ответ пользователю
- Если все факты ✅ — обычный ответ
- Если есть ❌ — скорректировать и перепроверить
- Если есть ❓ — указать «требует уточнения»
- Если verifier недоступен — ответ с «⚠️ без аудита»

## Пример
**Вопрос:** «Какие выплаты положены инвалиду II группы в Москве?»
**Шаг 1:** Категория = инвалидность. Регион = Москва.
**Шаг 2:** Читать `federal-law.md` (181-ФЗ, инвалиды) + `payments-table.md` (ЕДВ II гр.) + `moscow-region.md`
**Шаг 3:** Факты:
- Факт 1: Инвалиды II группы получают ЕДВ по 181-ФЗ
- Факт 2: Размер ЕДВ зависит от НСУ (натурой или деньгами)
- Факт 3: В Москве — бесплатный проезд + региональная доплата
**Шаг 4:** → social-verifier
**Шаг 5:** ✅ → ответ пользователю

## Ошибки
- Не гадать размер выплат — если нет точной цифры в БЗ, указать «размер уточняется»
- Не смешивать категории — если вопрос про ВБД, не читать раздел про инвалидов
- Если verifier молчит >30 сек — считать недоступным, ответить с пометкой
