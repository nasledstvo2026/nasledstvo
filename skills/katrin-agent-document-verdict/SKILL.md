# SKILL.md — Вердикт юриста к документам katrin-agent

## Назначение
При генерации документов katrin-agent обязан получить заключение от auditor-agent и включить его в итоговый ответ.

## Правило

1. **После создания документа** katrin-agent передаёт черновик auditor-agent.
2. **auditor-agent проверяет:**
   - Корректность ссылок на НПА
   - Соответствие статусу закупки
   - Арифметику (НМЦК, обеспечение, НДС)
   - Обязательные разделы
   - Потенциальные риски
3. **Вердикт auditor-agent:**
   ```
   🔍 Заключение auditor-agent:
   • Риски: [HIGH / MEDIUM / LOW]
   • Замечания: [список, если есть]
   • Статус: [одобрено / доработка / не одобрено]
   ```
4. **katrin-agent включает вердикт** в финальное сообщение. Если замечания — сначала исправляет, затем повторная проверка.

## Формат финального ответа
```
📄 Документ: [название].docx

[описание: параметры, позиции, сумма]

🔍 Заключение auditor-agent:
• Риски: LOW
• Замечания: нет
• Статус: ✅ одобрено
```
