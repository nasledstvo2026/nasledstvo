## Архитектура проекта

### Источники данных (по приоритету)
1. **ISW — Russian Offensive Campaign Assessment** (web_fetch)
2. **The Guardian / Al Jazeera** — геополитика
3. **Meduza / The Moscow Times** — внутренняя политика РФ
4. **Коммерсантъ / РБК** — экономика РФ, санкции
5. **VTsIOM / Levada** — социология, рейтинги
6. **government.ru / kremlin.ru** — официальные заявления

### Разделы дайджеста
1. **Геополитика / НАТО** — угрозы провокаций
2. **СВО / Украина** — динамика фронта, переговоры
3. **Мобилизация** — открытая / скрытая, реестр
4. **Экономика / Санкции** — рубль, бюджет, инфляция
5. **Внутренняя политика** — рейтинги, элиты, выборы

### Система оценок вероятности
- `prob-high` 🔴 — высокая (1-3 мес)
- `prob-mid` 🟡 — средняя (6-12 мес)
- `prob-low` 🟢 — низкая (год+)
- `prob-na` ⚪ — неопределённо

### Формат дайджеста

## Принципы
1. Только факты из проверенных источников. Без домыслов.
2. Каждое утверждение — с источником + датой.
3. Вероятность — всегда с обоснованием.
4. Дедлайн: пн/чт 08:00 — публикация дайджеста.
5. Раз в 3 часа — health-check доступности источников.
6. При сбоях — уведомление Кириллу (@Kirill_syst).
7. Старые данные — удалять, заменять актуальными.
