   - ISW Russian Offensive Campaign Assessment (web_fetch) — **основной источник по СВО**

4. **Опроси качественные СМИ (tier 3):**
   - The Guardian — Ukraine/Russia section

   - Al Jazeera — Russia/Ukraine section (вспомогательный, с пометкой source_type: secondary)

6. **Фильтрация:** Ранжируй по иерархии:
   - **Tier 1 (первичные данные):** принимаются без ограничений, наивысший приоритет

7. **Сохрани результат** в memory/osint/osint-raw-data.json.
   Формат JSON: массив объектов {source, date, fact, relevance, source_type, tier}
   source_type: "primary" | "secondary"

   relevance: "svo" | "mobilization" | "nato" | "economy" | "fuel"

8. **Запиши в лог** memory/osint/osint-log.md:
   - "[YYYY-MM-DD HH:MM] Фаза A: ISW — успех/ошибка"

   - "[YYYY-MM-DD HH:MM] Фаза A: BBC — успех/ошибка"
   - "[YYYY-MM-DD HH:MM] Фаза A: СПбМТСБ — найдено N/данные"
   - "[YYYY-MM-DD HH:MM] Фаза A: ГАС Правосудие — данные/ошибка"
   - "[YYYY-MM-DD HH:MM] Фаза A: SearXNG — найдено N результатов, total фактов"
   - "[YYYY-MM-DD HH:MM] Фаза A: сохранено N фактов в raw-data"
```

2. **Война с Европой/НАТО** — угрозы, провокации, союзники
   - Цвета: Высокая→🔴, Средняя→🟡, Низкая→🟢
3. **Мобилизация** — открытая / скрытая, реестр, социология
   - Цвета: Высокая→🔴, Средняя→🟡, Низкая→🟢
4. **Топливный кризис в РФ** — цены, дефицит, НПЗ, санкции, запасы
   - Цвета: Высокая→🟢, Средняя→🟡, Низкая→🔴 (как у СВО — зелёный = хорошо)
5. **Сводка ключевых событий** — краткий дайджест без оценок

## Система оценок вероятности (контекстно-зависимые цвета)
### СВО (зелёный = завершение — хорошо)
- `prob-low` 🟢 — Высокая (>70%) = СВО завершится
- `prob-mid` 🟡 — Средняя (40-70%) = неопределённо
- `prob-high` 🔴 — Низкая (<40%) = СВО продолжится

### Война с НАТО / Мобилизация (красный = событие произойдёт — плохо)
- `prob-high` 🔴 — Высокая (>70%) = начнётся
- `prob-mid` 🟡 — Средняя (40-70%) = неопределённо
- `prob-low` 🟢 — Низкая (<40%) = маловероятно

### Топливный кризис (зелёный = кризис завершается — хорошо, как у СВО)
- `prob-low` 🟢 — Высокая (>70%) = кризис завершается
- `prob-mid` 🟡 — Средняя (40-70%) = неопределённо
- `prob-high` 🔴 — Низкая (<40%) = кризис затягивается

### Общие правила
- При противоречивых данных — только Средняя (🟡)
- CSS-классы: prob-high, prob-mid, prob-low — не меняются
- Дисклеймер на странице: «Для СВО цвета инвертированы»

## Принципы
1. **Первичные данные > интерпретации.** Заявления ведомств сверяются с биржевыми котировками, судебными базами и трекерами. Данные из Tier 1 имеют приоритет над Tier 2-3.
2. Только факты из проверенных источников. Без домыслов.
3. Каждое утверждение — с источником + датой (связка source + date обязательна).
4. Вероятность — всегда с обоснованием и минимум 2 фактами на оценку.
5. Пропагандистские и анонимные источники — отсекать на этапе Фазы B.
6. **Secondary — только с перекрёстным подтверждением от Tier 1-3.** Неподтверждённые не участвуют в оценке.
7. Лог (osint-log.md) — обнуляется при старте Фазы A, заполняется всеми фазами.
8. Дедупликация — по связке источник + дата + утверждение (osint-seen.md).
9. При сбоях — failureAlert после 2 ошибок → уведомление Кириллу, cooldown 1 час.
10. **Цвета контекстно-зависимые:** СВО инвертирована, остальные — стандартные.
