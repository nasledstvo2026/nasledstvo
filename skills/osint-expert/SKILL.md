```

   Доверенные: ISW, The Guardian, BBC, Meduza, Коммерсантъ, РБК, официальные сайты госорганов (kremlin.ru, government.ru, pravo.gov.ru), Рейтер

## Система оценок вероятности (контекстно-зависимые цвета)
### СВО (зелёный = завершение — хорошо)
- `prob-low` 🟢 — Высокая (>70%) = СВО завершится
- `prob-mid` 🟡 — Средняя (40-70%) = неопределённо
- `prob-high` 🔴 — Низкая (<40%) = СВО продолжится

### Война с НАТО / Мобилизация (красный = событие произойдёт — плохо)
- `prob-high` 🔴 — Высокая (>70%) = начнётся
- `prob-mid` 🟡 — Средняя (40-70%) = неопределённо
- `prob-low` 🟢 — Низкая (<40%) = маловероятно

### Общие правила
- При противоречивых данных — только Средняя (🟡)
- CSS-классы: prob-high, prob-mid, prob-low — не меняются
- Дисклеймер на странице: «Для СВО цвета инвертированы»

## Принципы
1. Только факты из проверенных источников. Без домыслов.
2. Каждое утверждение — с источником + датой (связка source + date обязательна).
3. Вероятность — всегда с обоснованием и минимум 2 фактами на оценку.
4. Пропагандистские и анонимные источники — отсекать на этапе Фазы B.
5. **Al Jazeera — только с перекрёстным подтверждением от доверенного источника.** Факты из Al Jazeera без подтверждения не участвуют в оценке.
6. Лог (osint-log.md) — обнуляется при старте Фазы A, заполняется всеми фазами.
7. Дедупликация — по связке источник + дата + утверждение (osint-seen.md).
8. При сбоях — failureAlert после 2 ошибок → уведомление Кириллу, cooldown 1 час.
9. **Цвета контекстно-зависимые:** СВО инвертирована, остальные — стандартные.
