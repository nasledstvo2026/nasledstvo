# Время Название Статус Промпт Доп. требования
1 08:30 Сводка жалоб (основная)
announce → Катя deepseek-v4-flash 		OK
~92 сек		 Сделай сводку жалоб по наследству в банках РФ за вчерашний день (4 площадки: banki.ru, otzovik.com, pikabu.ru + sravni.ru). Не публикуй HTML. Используй SearXNG (localhost:8888/search) — до 6 запросов за раз. Результаты запиши в katya-stats-data.md. Если жалоб 0 — не пиши строку статистики. Отвечай ТОЛЬКО текстом сводки. —
2 08:45 Сводка жалоб (резерв)
announce → Катя deepseek-chat 		OK Резервный запуск. Аналогично основной задаче, но модель deepseek-chat. Срабатывает, если основная задача выдала ошибку. —
3 08:50 Статистика жалоб
без доставки deepseek-v4-flash 		ERROR
LLM request failed		 Обнови stats-inheritance.html на основе шаблона и katya-data.json. Рассчитай: дата дня, статистика месяца, годовая разбивка по банкам, таблица по месяцам, хронология. Опубликуй через publish-report.sh. Ответ в чат не нужен. —

Общие параметры

  • Failure alert: после 2 ошибок → Кириллу (cooldown 1 час)
  • Таймаут: 900 секунд (15 минут)
  • Fallback: deepseek-v4-flash → deepseek-chat
  • Публикация: GitHub Pages через publish-report.sh
