Архитектура

1. Источники данных
Используется трёхуровневая система:

🟢 LegalMCP (MCP-сервер) — специализированный юридический ИИ-сервер по праву РФ. Подключается одной строкой конфигурации через MCP-протокол (Anthropic). Содержит 13 кодексов РФ, 9 000+ федеральных законов, постановления правительства, судебную практику. 7 инструментов: семантический поиск, точное извлечение статей, граф связей, хронология редакций с diff, карта покрытия, мониторинг изменений, feedback loop. 100 запросов/мес бесплатно.
🟡 pravo.gov.ru — эталонная верификация (Указ Президента № 90). Официальный портал правовой информации.
🔵 Аналитические обзоры — контекст и практика (pro-ability.ru, fz44.org, medcub.ru).
2. Как работает агент
🎯 Цель: давать Катрин точные, юридически выверенные ответы по 44-ФЗ, 224-ФЗ, закупкам ЛП и МИ — с актуальными ссылками на закон, без галлюцинаций LLM.

Каждые пн/ср/пт в 09:30 запускается cron-задача:

1. Подключается к LegalMCP через MCP-протокол
2. Делает 7 семантических запросов по ключевым документам
3. Сохраняет конспект изменений в базу знаний агента (knowledge/katrin/weekly-update.md) — чтобы при каждом обращении Катрин не гонять LegalMCP снова, а сразу иметь свежий контекст
4. При обращении пользователя — перед ответом читает актуальную базу
5. Если изменения критичные — алерт в Telegram
🛠 Skill: katrin-tender-expert
Для Катрин зафиксирован отдельный постоянный навык, который гарантирует:

• Ответы от лица поставщика (она — поставщик ЛП/МИ, не заказчик)
• Учёт специфики: лекарства, медизделия, Честный ЗНАК, ПП РФ №620
• При ответах читает базу знаний + актуальные изменения из LegalMCP
• Всегда указывает источник (LegalMCP → pravo.gov.ru → аналитика)
• При сомнении — честно, с предложением проверить на pravo.gov.ru
• Skill применён и активен с 28.06.2026
3. Контроль качества
Обратная связь (Feedback Loop): при обнаружении устаревших данных отправляется тикет на обновление краулера LegalMCP.
Skill-сессия: для Катрин зафиксирован отдельный промпт — агент знает, что рекомендации даются от лица поставщика, с учётом специфики ЛП/МИ, нацрежима, маркировки и ЕИС.
База знаний: аккумулирует основные положения 44-ФЗ, 224-ФЗ, ПП № 620, ПП № 1875 — не нужно каждый раз искать с нуля.
4. Бесплатный лимит
LegalMCP — 100 запросов/мес бесплатно. Текущий расход: 84 запроса/мес (7 запросов × 3 раза/нед × 4 нед). Запас 16%. При превышении — переход на Pro (9 900 ₽/мес) или интеграция с платными API (КонсультантПлюс / Гарант).
