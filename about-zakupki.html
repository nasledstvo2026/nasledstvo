ФармТендер

Мониторинг госзакупок лекарственных препаратов и медицинских изделий
v1.0 · Июль 2026 · Продакшен
ФармТендер — это система ежедневного мониторинга государственных закупок лекарственных препаратов и медицинских изделий по 44-ФЗ и 223-ФЗ. Прямой парсинг zakupki.gov.ru, визуализация данных на сайте. Экономия времени тендерного специалиста — до 2 часов в день.

Как это работает

Один cron-парсер → JSON → страница на GitHub Pages:

07:00
🤖 Парсинг
zakupki.gov.ru
07:20
🏥 Health-check
проверка целостности
07:30
📋 Верификация
44-ФЗ + 223-ФЗ

Киллер-фичи

📡 1. Прямой парсинг ЕИС

Данные напрямую с zakupki.gov.ru, без посредников. Никаких агрегаторов с задержками.

💊 2. Ключевые запросы (ЛП, МИ, фармацевтика)

5 целевых запросов: лекарственные средства, медизделия, препараты, фармпродукция.

🩺 3. Health-check

Автоматическая проверка целостности данных после каждого парсинга.

📊 4. Визуализация на сайте

Таблица с фильтром по закону, статусу, заказчику, цене. Клиентский fetch JSON.

Все фичи продукта

📡 Парсинг

  • 5 поисковых запросов
  • До 100 записей за раз
  • Схлопывание дублей по номеру
  • Сортировка по дате

🏥 Мониторинг

  • Health-check после парсинга
  • Верификация 44-ФЗ/223-ФЗ
  • Автообновление будни 07:00

📊 Визуализация

  • Таблица: закон, номер, статус, заказчик
  • Цена, объект закупки, дата
  • Клиентский fetch JSON

🔄 Отказоустойчивость

  • План отката за 5 минут
  • Резервный парсинг
  • Проверка HTTP 200
