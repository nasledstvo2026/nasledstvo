# Test Cases: Иконка ФармТендеролог на главной

## TC-01: Иконка отображается
1. Открыть https://nasledstvo2026.github.io/nasledstvo/index.html
2. Найти плашку «ФармТендеролог» (самая нижняя)
3. **Ожидание:** Слева от заголовка видна иконка 32×32px

## TC-02: Размер плашки не изменился
1. Открыть index.html
2. Сравнить высоту плашки ФармТендеролог с соседними (AURA)
3. **Ожидание:** Высота примерно равна другим `.card.aura` плашкам (±5px)

## TC-03: Иконка не ломает ссылку
1. Кликнуть по плашке ФармТендеролог
2. **Ожидание:** Переход на zakupki.html (а не dead click)

## TC-04: Адаптивность (мобильные)
1. Открыть index.html на ширине < 600px
2. **Ожидание:** Иконка не выпадает, заголовок не переносится на новую строку из-за иконки

## TC-05: Hover
1. Навести курсор на плашку ФармТендеролог
2. **Ожидание:** Hover-эффект работает (подсветка, тень), иконка не дёргается

## TC-06: Регрессия — остальные плашки
1. Проверить все остальные плашки на странице
2. **Ожидание:** Ни одна не изменилась визуально

## TC-07: assets доступен
1. Открыть https://nasledstvo2026.github.io/nasledstvo/assets/pharm-tenderolog-icon.png
2. **Ожидание:** HTTP 200, отдаётся PNG 32×32
