📌 Легенда прототипа

Это демонстрационный макет итоговой таблицы. В реальной работе:

  • данные загружаются из базы мер, сформированной по результатам поиска и анализа НПА
  • источники — реальные ссылки на конкретные пункты документов
  • статусы «новая» / «изменена» / «не действует» присваиваются автоматически при сверке
  • для ВБД указывается подкатегория, для инвалидов — группа, для СВО — получатель
  • суммы индексируются в соответствии с актуальными НПА
  • региональные меры отображаются с привязкой к конкретному региону
