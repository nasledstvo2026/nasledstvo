# Мамкин аналитик

Мультиагентный сервис в Telegram для формирования бизнес-требований (БТ) высокого уровня зрелости.

## Архитектура

Сервис использует 4 специализированных AI-агента (на базе OpenClaw + DeepSeek API):

| Агент | Назначение |
|-------|-----------|
| **questioner** (опросчик) | Задаёт вопросы по 7 блокам шаблона БТ, адаптируя глубину (L1→L2→L3) |
| **compiler** (составитель) | Формирует текст БТ по утверждённому шаблону |
| **controller** (контролёр) | Проверяет каждый раздел на полноту и глубину |
| **generator** (генератор) | Создаёт DOCX и веб-версию, публикует на GitHub Pages |

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <url>
cd mamkin-analitik

# 2. Настроить переменные окружения
cp .env.template .env
# Отредактировать .env: указать DEEPSEEK_API_KEY, TELEGRAM_BOT_TOKEN и др.

# 3. Установить зависимости
pip install python-docx

# 4. Запустить
openclaw start
```

## Шаблон БТ

Сервис использует неменяемый шаблон из 7 блоков с динамической глубиной опроса (L1/L2/L3):

1. **Контекст и проблема** (контекст, проблема, последствия бездействия)
2. **Цель и измерение** (бизнес-цель, критерии успеха)
3. **Стейкхолдеры и влияние** (заинтересованные стороны, их интересы)
4. **Решение** (функции, сценарии, граничные случаи)
5. **Требования к качеству и ограничения**
6. **Приёмка** (критерии приёмки)
7. **Риски и митигации**
