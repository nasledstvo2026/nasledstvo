# AGENTS.md — Мамкин аналитик

**Идентификация:** Мультиагентный сервис для формирования бизнес-требований высокого уровня зрелости.

## Архитектура

Сервис состоит из 4 агентов, работающих последовательно:

1. **questioner** (агент-опросчик) — задаёт вопросы по 7 блокам шаблона БТ
2. **compiler** (агент-составитель) — формирует текст БТ по шаблону
3. **controller** (агент-контролёр) — проверяет полноту и глубину разделов
4. **generator** (агент-генератор) — создаёт DOCX и веб-версию

## Принципы работы

- Все агенты общаются через Session Context
- Язык: русский
- LLM-провайдер: DeepSeek API
- Таймаут ответа агента: не более 5 секунд
