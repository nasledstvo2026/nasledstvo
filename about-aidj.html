AI DJ

Интеллектуальная система автоматического сведения музыки
v1.0 · Июль 2026 · Продакшен
AI DJ — это ИИ-диджей, который автоматически определяет BPM и тональность треков с помощью librosa, синхронизирует их и создаёт профессиональные beat-synced миксы. 100+ треков, веб-плеер с Club EQ (Web Audio API), Cloudflare-туннель для прямой потоковой передачи.

Как это работает

Два компонента: Flask-сервер на VPS и веб-интерфейс на GitHub Pages:

🎛 Сервер (Flask)

  • librosa — BPM/key детекция
  • ffmpeg — beat-synced crossfade
  • Cloudflare Tunnel — HTTPS
  • CRUD треков и сетов

🌐 Клиент (GitHub Pages)

  • djset.html — создание сетов
  • aidj-player.html — плеер с EQ
  • aidj-delete.html — удаление
  • Club EQ через AudioContext

Киллер-фичи

🎵 1. BPM/Key детекция (librosa)

Автоматическое определение темпа и тональности. Не нужно размечать треки вручную.

🔄 2. Beat-synced crossfade

Переходы строго по сетке BPM. Профессиональное звучание без редактирования.

🎚 3. Club EQ через Web Audio API

Цепочка BiquadFilter: lowshelf +4dB, peak +3dB на 250Hz/12kHz, cut на 1k/4kHz. Без слайдеров.

🔒 4. Cloudflare туннель — без прокси

Прямой HTTPS-доступ к серверу. Никаких прокси-серверов, минимум latency.
