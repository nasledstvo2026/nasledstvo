## 1. Концепция продукта

«ФармТендеролог» — интеллектуальный юридический ассистент для участников закупок лекарственных препаратов (ЛП) и медицинских изделий (МИ) по 44-ФЗ и 224-ФЗ.

Продукт автоматизирует создание юридически корректных документов (допсоглашения, закупочная документация) с многоуровневой проверкой на соответствие нормативной базе, практике УФАС и реестрам предельных цен. А также консультирует по предметной области на уровне эксперта.

## 4. Пользовательские сценарии (User Stories)

1. **Как специалист по тендерным продажам и закупкам**, я хочу отправить запрос на разработку документа — и получить готовый .docx, чтобы не тратить время на написание вручную.
2. **Как специалист по тендерным продажам и закупкам**, я хочу чтобы документ автоматически проверялся на соответствие 44-ФЗ, чтобы исключить ошибки.
3. **Как специалист по тендерным продажам и закупкам**, я хочу чтобы проверка включала практику УФАС по аналогичным кейсам, чтобы не получить жалобу.
4. **Как специалист по тендерным продажам и закупкам**, я хочу чтобы при изменении цены препарата система сверялась с реестром предельных цен ЖНВЛП.
5. **Как специалист по тендерным продажам и закупкам**, я хочу консультироваться с агентом в чат-боте Telegram по тендерным вопросам, чтобы получать быстрые ответы без ожидания юриста.

## 5. Бизнес-ценность

| Метрика | Ожидаемый эффект |
|---------|-----------------|
| Время на подготовку допсоглашения | Сокращение с ~2-3 часов до 15-20 минут |
| Доля документов с юр. ошибками | Снижение до <5% (с ручных ~30-50%) |
| Риск жалобы в УФАС | Минимизация за счёт сверки с практикой |
| Скорость реакции на изменения в НПА | Автоматическое обновление знаний (LegalMCP) |

## 7. Ограничения

- Бесплатный лимит LegalMCP — 100 запросов/месяц
- Открытые API УФАС, ЕИС, ЖНВЛП — требуется исследование на доступность
- Формат документов — только .docx (Microsoft Office), никакого Markdown, никаких .md-шаблонов
- Infra: GitHub Pages + VPS + Cloudflare Tunnel (без кастомного домена)

## 8. Критерии успеха (MVP)

- [ ] katrin-agent отвечает на вопросы по 44-ФЗ/224-ФЗ с цитатами НПА
- [ ] katrin-agent генерирует .docx по запросу
- [ ] auditor-agent проверяет документ на соответствие нормам
- [ ] katrin-agent вносит исправления в документ на основании замечаний auditor-agent
- [ ] Цикл «генерация → аудит → исправление → отдача» работает в один проход
- [ ] Telegram-маршрутизация: сообщения от специалиста приходят только katrin-agent
