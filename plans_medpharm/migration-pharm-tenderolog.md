1. **Ничего не ломать для существующих пользователей.** Все остальные (Лена, Данил, Катя, Роза, Ирина) продолжают работать как есть

## 4. Риски и mitigation

| Риск | Вероятность | Влияние | Mitigation |
|------|------------|---------|------------|
| Telegram-маршрутизация сломает другие каналы | Низкая | Высокое | Тестировать на изолированном окружении, готовый откат |
| LegalMCP лимит 100/мес не хватит на 2 агентов | Средняя | Среднее | Мониторинг расхода. Если не хватает — платный тариф или оптимизация |
| УФАС/ЕИС/ЖНВЛП не имеют открытых API | Высокая | Среднее | Fallback: web_search + web_fetch для ручного аудита |
| sessions_send не поддерживает нужный протокол | Низкая | Среднее | Fallback: katrin-agent даёт команду auditor-agent через общий канал (main) |
| auditor-agent не успевает за 120 сек | Средняя | Низкое | Увеличить таймаут. Fallback «без аудита» |
| katrin-agent генерирует невалидный .docx | Низкая | Среднее | Проверка python-docx. Если битый → retry |
| Специалист не получает ответ в Telegram | Низкая | Высокое | Немедленный откат маршрутизации на main |
