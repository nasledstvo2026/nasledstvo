💡 Что такое MedHelp

MedHelp — интеллектуальный ассистент для врачей лучевой диагностики (КТ, МРТ, ПЭТ/КТ). Разработан для врача-рентгенолога Марии Комаровой как помощник в рутинной работе: подготовка заключений, поиск актуальных протоколов исследований, верификация ссылок на клинические рекомендации.

🔬 Возможности

📋
Протоколы исследований
Помощь в подготовке заключений: техника, находки, заключение. Рекомендации по дозиметрии (НРБ-99/2009).
📖
Клинические рекомендации
Поиск и цитирование актуальных протоколов Минздрава РФ и РОРР. Дословные цитаты с указанием раздела, пункта, даты утверждения.
✅
Верификация фактов
Каждый факт проверяется по двум независимым источникам. Вердикты: ✅ ❌ 🔄 ❓
🌐
Международные источники
Radiopaedia.org, ACR Appropriateness Criteria, PubMed. Используются при отсутствии российских протоколов.

⚙️ Как это работает

Двухагентная архитектура

medhelp-agent — фронтальный ассистент. Общается с врачом в Telegram, помогает с формулировками, ищет протоколы, формирует ответ.

medcheck-agent — изолированный верификатор. Проверяет каждый клинический факт по источникам и возвращает вердикт.

Перед выдачей ответа medhelp-agent отправляет факты в medcheck-agent. Если верификация провалена или источники недоступны — агент выдаёт ответ с пометкой ❓ и рекомендацией сверить с первоисточником.

Вердикты верификации

✅ Подтверждено ❌ Противоречие 🔄 Расхождение источников ❓ Нет данных в РФ

🛠 Техническая основа

  • Платформа: OpenClaw (изолированные сессии агентов)
  • Модель: DeepSeek Chat (deepseek/deepseek-chat)
  • Источники (РФ): cr.minzdrav.gov.ru (Минздрав), rorr.ru (РОРР)
  • Источники (междунар.): Radiopaedia.org, ACR, PubMed
  • Нормативы: НРБ-99/2009 (СанПиН) по дозиметрии
  • Архитектура: двухагентная, agent-to-agent, peer binding

🚫 Что ассистент НЕ делает

  • Не ставит медицинские диагнозы
  • Не описывает снимки (AI-анализ изображений)
  • Не назначает лечение и не выписывает рецепты
  • Не подписывает заключения (финальный документ — ответственность врача)
  • Не интегрируется с PACS/RIS-системами

📱 Инструкция для Марии

Как пользоваться MedHelp:

  1. Открой Telegram — пиши в тот же чат, что и обычно
  2. Задавай вопросы по протоколам КТ, МРТ, ПЭТ/КТ свободным текстом
  3. Примеры запросов:
    «Какой протокол КТ ОГК при подозрении на рак лёгкого?»
    «Показания для КТ с контрастом при ТЭЛА»
    «Нормы доз на КТ ОГК»
  4. Дождись ответа (обычно 30–60 сек) — ассистент проверяет факты по источникам

Вердикты в ответах:

  • ✅ — данные подтверждены Минздравом РФ или РОРР
  • 🔄 — источники расходятся, решение за врачом
  • ❓ — нет российского стандарта, информация из международных источников. Рекомендуется сверить с первоисточником

⚠️ MedHelp — ассистент, а не замена врачу. Протоколы и дозиметрию сверяй с клиническими рекомендациями.
