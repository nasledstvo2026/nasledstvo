# База диалогов — Адвокат

Хронологический лог переписки с матерью Василисы.
Конфиденциально. Только для Кирилла и Лунта.

## Формат записи

```yaml
---
date: YYYY-MM-DD
channel: WhatsApp/Telegram/устно
type: тип_записи
tags: [тег1, тег2]
---
```

Типы: `отказ_в_общении`, `манипуляция`, `обесценивание`, `газлайтинг`, `SOS`, `соглашение`
