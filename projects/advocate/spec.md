## 7. База диалогов (dialogue-log)

### 7.1. Назначение
Хронологический лог переписки с матерью Василисы. Ключевой актив для:
- Распознавания её персональных паттернов абьюза (PAS, газлайтинг, обесценивание)
- Хронологии нарушений для суда/органов опеки
- Отслеживания эскалации (учащение отказов, смена тактики)
- Формирования персонализированных ответов Адвоката (ссылки на её конкретные слова)

### 7.2. Формат хранения
Markdown + YAML frontmatter. Одна запись — один эпизод.

```yaml
---
date: 2026-07-12
channel: WhatsApp
type: отказ_в_общении
tags: [PAS, обесценивание, ст66]
---
**Она:**
«Ты ей не нужен, она сама не хочет с тобой гулять, отстань от нас»

**Я:**
«Согласно ст. 66 СК РФ я имею равное право на общение. Предлагаю письменный график»

**Анализ:**
🧠 Обесценивание + газлайтинг
📋 Нарушение ст. 66 СК РФ — препятствие общению
📌 Доказательство: скриншот сохранён
```

### 7.3. Типы записей
| type | Описание |
|------|----------|
| `отказ_в_общении` | Прямой запрет встречи / препятствие общению |
| `манипуляция` | Давление через ребёнка, угрозы |
| `обесценивание` | «Ты плохой отец / мало платишь / не нужен» |
| `газлайтинг` | «Ты неправильно помнишь», «она не хочет», «она боится» |
| `SOS` | Острый конфликт, срочно нужен ответ |
| `соглашение` | Предложение/обсуждение графика встреч |

### 7.4. RAG
**Не нужен на старте.** Внедрить, когда объём базы превысит 30–50 записей. Тогда — vector storage на VPS, semantic search для агента.

Файл: `projects/advocate/dialogue-log.md`

### 7.5. Безопасность
- Файл ТОЛЬКО в проектной папке
- НЕ публикуется на GitHub Pages
- НЕ попадает в cron-отчёты
- НЕ синхронизируется с публичными страницами

## 9. Переговорная тактика — экспертиза Адвоката

### 9.1. Контекст
Асимметричные переговоры. Мать провоцирует, манипулирует, давит эмоционально. Кирилл не может ругаться при ребёнке. Цель — не убедить мать, а зафиксировать свою позицию и её нарушения.

### 9.2. Техники

#### Серая скала (Grey Rock)
Стать скучным, неэмоциональным, предсказуемым. Ответы короткие, сухие, без крючков.
- Она: «Ты ужасный отец!»
- Ты: «Ст. 66 СК РФ — предлагаю письменный график встреч.»
- Она: «Ты мне надоел!»
- Ты: «График жду на согласование до пятницы.»

#### Заезженная пластинка (Broken Record)
Повтор одной юридической позиции, не переключаясь на её темы.
- Она: «Она не хочет» → «Право на общение зафиксировано в ст. 66 СК РФ. Предлагаю график.»
- Она: «Ты ей чужой» → «Ст. 66 СК РФ — предлагаю график.»
- Она: «Ты не платишь» → «Алименты плачу исправно. Ст. 66 — предлагаю график.»

#### Перевод в письменный формат
Если она начинает эмоциональный разговор — перевести в переписку.
- «Давай письменно. Мне нужна твоя чёткая позиция по графику встреч.»
- «Я фиксирую отказ. Фиксирую дату и время.»

#### Фиксация + переход к действию
- «Твой отказ зафиксирован. Повторю предложение: место/время/дата. Жду подтверждения до [дата].»
- «Фиксирую как препятствие общению. Предлагаю перевести диалог к соглашению.»

### 9.3. Эскалационная лестница

| Ступень | Что происходит | Что делает Кирилл | Кто участвует |
|---------|---------------|-------------------|---------------|
| 1 | Устные договорённости | Избегать. Всё в письменный вид | Только он и мать |
| 2 | Письменное предложение графика | Отправить проект соглашения (ст. 66 п. 2) | Только он и мать |
| 3 | Предложение + предупреждение | «Повторяю предложение. Ст. 66 — препятствие незаконно. Зафиксирую отказ.» | Переписка |
| 4 | Досудебная претензия | Официальное письмо с требованием определить график | Нотариус / заказное письмо |
| 5 | Органы опеки | Вызов опеки для фиксации нарушения | Опека, мать, Кирилл |
| 6 | Судебный иск | Иск об определении порядка общения | Суд, адвокат, опека |

**Правило:** не прыгать через ступени. Переходить только когда исчерпана предыдущая.

### 9.4. Правила переписки (для Адвоката)

1. Если сообщение матери НЕ содержит конкретики (даты, предложения) → не вовлекаться. «Жду конкретное предложение по графику.»
2. Если сообщение содержит оскорбления → игнорировать содержание, повторить предложение. Не оправдываться, не доказывать.
3. Если сообщение содержит угрозу → «Фиксирую. Мои обязательства исполняются. Ответ по существу?»
4. Если сообщение содержит ложь → «Не соответствует действительности. Документы предоставлю по запросу суда/опеки.»
5. Если сообщение — просьба/конкретика → ответить по существу, чётко, с датами.

### 9.5. Чек-лист «готовность к разговору» (для Кирилла)
Перед тем как написать ей:
- [ ] Я не зол/не обижен? Если да → подождать час
- [ ] Моё сообщение содержит конкретику (даты, статьи, предложения)?
- [ ] Я готов к тому, что она проигнорирует/обесценит?
- [ ] Мой ответ не содержит оскорблений и эмоций?
- [ ] Я зафиксирую её ответ как доказательство?
