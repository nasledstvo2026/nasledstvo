# Бизнес-требования к проекту OSINT

**Проект:** OSINT-дайджест  
**Дата:** 04.07.2026  
**Версия:** 0.1  
**Статус:** Черновик

---

## 1. Предметная область

Сбор, анализ и представление информации по направлениям:
- СВО (специальная военная операция)
- Вторая волна мобилизации
- Война с Европой / НАТО (геополитические риски)

## 2. Описание проблемы

Люди встревожены возможной войной с Европой, мобилизацией и затяжным характером СВО. Поступает большое количество противоречивой информации. Нет инструмента, который позволял бы:

- Отфильтровывать недостоверные источники
- Собирать ключевые факты в единую картину
- Давать объективную оценку текущей ситуации

Из-за этого растёт тревога, а понять реальное положение дел — невозможно.

## 3. Описание цели

Создать инструмент, способный объективно (на основе фактов и аналитики данных) оценить и наглядно показать пользователю степень вероятности:

1. Завершения СВО
2. Начала войны с Европой
3. Начала второй волны мобилизации в РФ

## 4. Критерии достижения результата

4.1. На странице https://nasledstvo2026.github.io/nasledstvo/osint.html по каждому из трёх вопросов видна оценка вероятности события. Оценка подкреплена фактами, её подтверждающими.

4.2. Работа проекта OSINT не мешает работе всех остальных цифровых проектов цифровой экосистемы Лунт.

## 5. Источники данных

Формирование объективной оценки базируется на следующих категориях источников:

### 5.1. Официальные государственные данные и реестры
- **Портал правовой информации РФ** (pravo.gov.ru) — указы, законы о мобилизации, изменения в УК РФ
- **Единый реестр воинского учёта РФ** — статус запуска цифровых повесток и ограничений
- **Официальные ведомства РФ** — Минобороны (mil.ru), МИД (mid.ru)
- **Госорганы ЕС и НАТО** — пресс-релизы Еврокомиссии, оборонные ведомства стран Балтии, Польши, Германии, штаб-квартира НАТО
- **Органы власти Украины** — Минобороны (mil.gov.ua), ГУР, СНБО

### 5.2. Международные аналитические центры (Think Tanks)
- **ISW (США)** — ежедневные отчёты по оперативной обстановке на фронте, **основной источник**
- **RUSI (Великобритания)** — военная аналитика, разборы тактики, ВПК РФ и НАТО
- **OSW (Польша)** — мониторинг внутренней политики РФ, мобилизационных процессов
- **Rand Corporation / CSIS (США)** — стратегическое моделирование сценариев
- **SIPRI** — данные по поставкам оружия, расходам на ВПК, ядерному сдерживанию

### 5.2а. Качественные СМИ (доверенные)
- **Би-би-си (BBC)** — международная журналистика, верифицированные данные
- **The Guardian** — качественная аналитика геополитики
- **Meduza** — независимая журналистика по РФ
- **Коммерсантъ / РБК** — экономика РФ, официальная картина
- **Рейтер (Reuters)** — лента новостей, фактчекинг

### 5.2б. Вспомогательные источники (с перекрёстной верификацией)
- **Al Jazeera** — только факты, подтверждённые другим доверенным источником
- Остальные СМИ — только через перекрёстную проверку с 2+ доверенными источниками

### 5.3. Профильные OSINT-проекты и верификация данных
- **Oryx** — верификация потерь техники по фото/видео
- **GeoConfirmed / WarMapper** — геолокация, спутниковая верификация
- **Медиазона / Би-би-си** — верифицированные списки погибших
- **Grey Dynamics / Janes** — коммерческая разведка, оборонные риски

### 5.4. Спутниковый мониторинг и радарные данные
- **Sentinel Hub (Copernicus) / NASA FIRMS** — тепловые аномалии
- **Maxar Technologies / Planet Labs** — снимки высокого разрешения
- **Flightradar24 / ADSB Exchange** — мониторинг военной и правительственной авиации

### 5.5. Мониторинг инфопространства и соцсетей
- **Российский Telegram** — Z-каналы, «Рыбарь», «Colonelcassad»
- **Украинские каналы** — «DeepState», «Николаевский Ванёк»
- **Правозащитные** — «Идите лесом», «Школа призывника»
- **Закрытые сообщества** — чаты родственников мобилизованных, региональные паблики

### 5.6. Экономические индикаторы и ВПК
- **Росстат и ЦБ РФ** — инфляция, дефицит бюджета, загрузка промышленности
- **Kiel Institute (Ukraine Support Tracker)** — западная помощь Украине
- **ImportGenius / Trade Data Monitor** — обход санкций, компоненты двойного назначения

## 6. Периодичность обновления

Дайджест обновляется по расписанию: 06:00, 14:00, 20:00 МСК.

## 7. Формат визуализации

Оценка вероятности события отображается в формате с **контекстно-зависимыми цветами**:

### 1. Вероятность завершения СВО
Чем выше вероятность — тем лучше для пользователя. Зелёный = хорошо (СВО завершится).

| Уровень | Цвет | Значение |
|---------|------|----------|
| Высокая | 🟢 Зелёный | СВО завершится / заморозится |
| Средняя | 🟡 Жёлтый | Неопределённая ситуация |
| Низкая | 🔴 Красный | СВО продолжится / эскалация |

### 2. Вероятность войны России с Европой/НАТО
Чем выше вероятность — тем хуже. Красный = плохо (война начнётся).

| Уровень | Цвет | Значение |
|---------|------|----------|
| Высокая | 🔴 Красный | Война начнётся — тревожный сигнал |
| Средняя | 🟡 Жёлтый | Риск есть, но неопределённо |
| Низкая | 🟢 Зелёный | Война маловероятна |

### 3. Вероятность второй волны мобилизации в РФ
Чем выше вероятность — тем хуже. Красный = плохо (мобилизация будет).

| Уровень | Цвет | Значение |
|---------|------|----------|
| Высокая | 🔴 Красный | Мобилизация начнётся |
| Средняя | 🟡 Жёлтый | Риск есть, но неопределённо |
| Низкая | 🟢 Зелёный | Мобилизация маловероятна |

## 8. Критерии объективности

Объективность оценки обеспечивается комбинацией:
- **Количество фактов** — оценка базируется на достаточном массиве данных из разных источников
- **Качество верификации** — LLM и агент-верификатор отвечают за перекрёстную проверку фактов из независимых источников
- **Мультиагентная архитектура** — разделение ролей сбора, анализа и верификации

Задача архитектуры: минимизировать влияние единичного источника на итоговую оценку.

## 9. Риски проекта

Каждый из следующих пунктов может пойти не так:

| Риск | Описание |
|------|----------|
| Недоступность источников | Часть источников (ISW, спутниковые данные) может быть недоступна или заблокирована |
| Противоречивые данные | Разные источники дают взаимоисключающую информацию |
| Технические сбои | Ошибки в cron-задачах, сбои LLM API, проблемы с git push |
| Ошибка LLM | Модель может неверно интерпретировать данные или пропустить важный контекст |
| Устаревание данных | Дайджест обновляется раз в сутки — ситуация может измениться быстрее |

## 10. Заинтересованные стороны

Целевая аудитория дайджеста — любые пользователи, заинтересованные в объективной оценке ситуации по СВО, мобилизации и рискам войны с Европой.
