## 1. Предусловия

- Система OpenClaw работает
- GitHub Pages доступен
- DeepSeek API отвечает
- Основные источники (ISW, The Guardian, Al Jazeera) доступны
- Файловая структура проекта создана: `projects/osint/breq.md`, `projects/osint/srs.md`, `memory/osint/`

| **Приоритет** | 🟡 Высокий |
