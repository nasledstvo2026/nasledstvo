> ⚠️ Этот файл — историческая справка. Актуальная архитектура: `knowledge/katrin/ARCHITECTURE.md`

## Описание
Цифровой продукт для Катрин (@Su_Ri_Mi) — консультации по 44-ФЗ, 223-ФЗ, закупкам лекарственных препаратов и медицинских изделий. Позволяет агенту отвечать с точными ссылками на закон.

## Архитектура (текущая)

```
[Катрин в Telegram]
       ↓ вопрос по 44-ФЗ/223-ФЗ/закупкам
[katrin-agent]
       ↓ перед ответом читает
┌─────────────────────────────────────┐
│  knowledge/katrin/zakupki-knowledge.md  │ ← база знаний (общие положения)
│  knowledge/katrin/weekly-update.md      │ ← актуальные изменения (последний cron)
│  knowledge/katrin/prompt-katrin.md       │ ← промпт сессии
└─────────────────────────────────────┘
       ↓ отправляет на верификацию
[auditor-agent] ← проверяет через consultant.ru / pravo.gov.ru
       ↓ возвращает вердикт
[katrin-agent] → ответ Катрин
```
