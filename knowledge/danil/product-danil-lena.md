# Продукт: Аналитика наследственных вкладов (СССР 1991)
## Для Данила (@danil, ID: 221828063)

## Описание
Цифровой продукт для анализа компенсаций по вкладам СССР 1991 года. Позволяет агенту отвечать на вопросы о размерах выплат, категориях граждан, индексациях и изменениях законодательства, используя базу знаний + LegalMCP + web_search.

## Архитектура

```
[Данил в Telegram]
       ↓ вопрос по вкладам 1991
[Лунт (OpenClaw)] ← Skill: danil-vklady-expert
       ↓ перед ответом читает
┌─────────────────────────────────────┐
│  knowledge/danil/vklady-knowledge.md   │ ← база знаний (методики, формулы, таблицы)
│  memory/danil-vklady-mon.md            │ ← новости за понедельник
│  memory/danil-vklady-thu.md            │ ← новости за четверг
└─────────────────────────────────────┘
       ↓ для точной статьи закона
[LegalMCP (MCP-сервер)] → законы (73-ФЗ, ПП №136/812/152)
[SearXNG] → новости
[pravo.gov.ru] → верификация
```

## Источники (по приоритету)
1. **LegalMCP** — законы и постановления (73-ФЗ, ПП №136, №812, №152)
2. **База знаний** — методики расчёта, коэффициенты, категории
3. **web_search** — новости и изменения
4. **pravo.gov.ru** — официальная верификация

## Цепочка восстановления при ошибках

### ❌ «LegalMCP не отвечает»
- Fallback: ответить из базы знаний (методики не меняются часто)
- Указать что законодательная часть не проверена

### ❌ «Новости не найдены (web_search)»
- В отчёте: «Нулевая медийная активность за период»
- Это штатная ситуация (тема малоактивна)

### ❌ «Коэффициенты устарели»
- Проверить ПП РФ последнее (ПП №152 продлён до 2026)
- Если новое постановление — обновить `vklady-knowledge.md`

# Продукт: Дайджест новостей по наследству + РЖД
## Для Лены (@lena, ID: 254785028)

## Описание
Цифровой продукт для мониторинга новостей и законодательства по наследству + облигации РЖД 1Р-37R. Позволяет агенту собирать дайджест новостей, итоги торгов и публиковать на GitHub Pages.

## Архитектура

```
[Лена в Telegram]
       ↓ получает дайджест 09:00 / РЖД 23:55
[Лунт (OpenClaw)] ← Skill: lena-news-expert
       ↓ собирает из
┌─────────────────────────────────────┐
│  knowledge/lena/news-knowledge.md     │ ← шаблоны, категории, дедупликация
│  knowledge/lena/rzd-knowledge.md      │ ← параметры РЖД 1Р-37R
│  memory/lena-news-daily.md            │ ← дневной дайджест
│  memory/lena-rzd-daily.md             │ ← итоги торгов
│  memory/lena-news-seen.md             │ ← история (дедупликация)
└─────────────────────────────────────┘
       ↓ поиск
[web_search] → новости (time_range=week)
[LegalMCP] → законодательные изменения
[MOEX ISS API] → данные по РЖД 1Р-37R
```

## Компоненты

| Компонент | Где | Назначение |
|-----------|-----|------------|
| База знаний новостей | `knowledge/lena/news-knowledge.md` | Категории, шаблоны, источники, дедупликация |
| База знаний РЖД | `knowledge/lena/rzd-knowledge.md` | Параметры выпуска, MOEX API, анализ |
| Дайджест | `memory/lena-news-daily.md` | Ежедневный дайджест |
| РЖД итоги | `memory/lena-rzd-daily.md` | Итоги будних торгов |
| Дедупликация | `memory/lena-news-seen.md` | Исторический список |
| Skill | `lena-news-expert` (applied 28.06) | Поведение агента |
| Отчёт | `report-lena.html` → GitHub Pages | Ежедневно 09:00 |
| РЖД отчёт | Telegram | Будни 23:55 |

## Источники (по приоритету)

**Для новостей:**
1. **web_search** — первичный поиск (3 запроса, time_range=week)
2. **прямые источники:** banki.ru, rg.ru, notariat.ru, duma.gov.ru
3. **LegalMCP** — только для верификации законодательных изменений

**Для РЖД:**
1. **MOEX ISS API** — прямой сбор (http://iss.moex.com/iss/...)
2. **rusbonds.ru** — справочно

## Алгоритм дедупликации
1. Проверить URL/заголовок в `memory/lena-news-seen.md`
2. Если найден → пропустить
3. Если новый → добавить в lena-news-seen.md
4. В дайджест: максимум 7 новостей (1-2 Главное + 3-5 Коротко)

## Цепочка восстановления при ошибках

### ❌ «MOEX ISS не отвечает»
- Проверить `https://iss.moex.com/iss/engines/stock/...`
- Fallback: rusbonds.ru (web_fetch)
- Указать что данные MOEX не получены

### ❌ «web_search не отвечает»
- Fallback: web_search (DDG)
- Если и DDG не отвечает — дайджест без новостей с пометкой

### ❌ «LegalMCP для законов устарел»
- Как у Катрин: указать дату кэша, рекомендовать pravo.gov.ru
