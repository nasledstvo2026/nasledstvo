# Меры социальной поддержки

**Статус:** 🟢 Активен  
**Дата создания:** 10.07.2026  
**Владелец продукта:** Команда Социальный консультант  

## Описание проекта

Справочно-информационный сервис по мерам социальной поддержки населения РФ. 
Содержит актуальный перечень федеральных и региональных льгот, выплат, компенсаций и субсидий,
а также порядок их получения.

## Состав продукта

- `socialhelp.html` — основной справочник мер поддержки (опубликован)
- `report-roza.html` — еженедельный обзор изменений в законах по пособиям
- `report-irina.html` — еженедельный обзор НПА по соцподдержке
