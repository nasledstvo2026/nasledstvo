🔄 Протокол взаимодействия

Межсессионная связь

katrin-agent → sessions_send(sessionKey="session:auditor", message=...) → auditor-agent auditor-agent → response (список замечаний / OK) → katrin-agent

Цикл аудита

  1. Генерация: katrin-agent создаёт черновик (python-docx)
  2. Отправка: katrin-agent шлёт auditor-agent (текст + метаданные)
  3. Аудит: auditor-agent проверяет по всем источникам
  4. Ответ: auditor-agent возвращает [замечание / риск / OK]
  5. Исправление: katrin-agent вносит правки
  6. Финал: katrin-agent отдаёт .docx Катрин + комментарий

Fallback

  • Таймаут auditor-agent (120 сек) → katrin-agent выдаёт «без аудита»
  • Ошибка auditor-agent → katrin-agent предлагает повторить

  • Межсессионная связь — только через Gateway (sessions_send)
