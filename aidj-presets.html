🔍 Исследуется
PO
Paul Oakenfold
Trance · Progressive · Goa · Ibiza pioneer
«Кинематографичное сведение» — треки звучат как единое полотно. Главные принципы:
① Long Blends (1.5–3 мин): входящий трек стартует на тихом интро без бочки поверх плотного финала предыдущего. Граница треков незаметна.
② Bass-подмена: на входе Bass = −∞, Mid/High на 9–10h. На 32–64 такте — быстрая подмена: Low старого → 9h, Bass нового → 12h. Резкий энергетический толчок.
③ Яма на Яму (Breakdown Mixing): кульминационные паузы двух треков накладываются. Ритм исчезает — переплетаются только эмбиент, вокал, синты. Максимальный драматизм.
④ Гармонический лифт: каждый следующий трек на ступень выше по Camelot Wheel (Am → Em). Подсознательное ощущение роста.
⑤ Кино-нарратив: Интро → Завязка → Кульминация → Эпилог. FX-сэмплы из фантастики для маскировки стыков.
🎯 Пример (Tranceport 1998): The Thrillseekers → Binary Finary.
00:00 Запуск Б (Bass=-∞, Mid 9h) · 00:45 Mid Б→12h, мелодии переплетаются · 01:30 Bass Swap (64 такта!) — резкая подмена баса · 02:15 Финал. Трек Б доминирует.
Crossfade: 1.5–3 min (layered) Tempo: lock tight ±0.5 EQ: Bass cut → sub on 32th bar Harmonic: Camelot +1 step up Structure: breakdown-on-breakdown Curve: exponential (slow swell) Source: The Goa Mix · Perfecto Fluoro · DJ Skills
