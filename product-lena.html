Консультант по наследству
Продукт для Лены · мониторинг новостей наследственного права · 4 агента

📦 Обзор продукта

Консультант по наследству — автоматизированная система мониторинга новостей наследственного права для пользователя Лены.

Продукт состоит из двух подсистем:

  • Cron-пайплайн — 3 изолированных агента, запускаемых по расписанию. Автоматически собирают, проверяют и публикуют данные без участия пользователя.
  • Чат-агент — 1 диалоговый агент, доступный через Telegram. Отвечает на запросы Лены в реальном времени, используя данные, подготовленные cron-пайплайном.

📰 Новостной пайплайн (cron)

Три агента, запускаемые ежедневно. Полный цикл: поиск → верификация → публикация.

⏰ 02:00 МСК search-agent → lena-raw.json (≤15 статей)
⏰ 02:10 МСК verify-agent → lena-verified.json (≤10 проверок)
⏰ 02:20 МСК html-agent → report-lena.html (≤7 новостей) → GitHub Pages + Telegram

Зазор 10 минут между этапами исключает race condition. Таймауты: search — 600s, verify — 300s, html — 300s. Модель: deepseek-v4-flash.
