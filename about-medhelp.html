MedHelp

Интеллектуальный ассистент для врача-рентгенолога
v1.0 · Июль 2026 · MVP
MedHelp — это ИИ-помощник для врачей лучевой диагностики. Помогает с протоколами КТ/МРТ/ПЭТ-КТ, клиническими рекомендациями и верификацией медицинских фактов через auditor-agent.

Как это работает

Два агента: основной и верификатор:

🩺 medhelp-agent — отвечает на вопросы врача, формирует протоколы, даёт клинические рекомендации.
🔍 auditor-agent — верифицирует каждый медицинский факт через клинические рекомендации, проверяет дозировки, противопоказания.

Киллер-фичи

✅ 1. Верификация через клинические рекомендации

Каждый факт проверяется auditor-agent перед выдачей врачу. Ни одной ошибочной рекомендации.

⚡ 2. Быстрые ответы

Вопрос → medhelp-agent → auditor-agent → ответ. Всё за секунды.

📋 3. Протоколы КТ/МРТ/ПЭТ-КТ

Структурированные протоколы описаний лучевых исследований.
