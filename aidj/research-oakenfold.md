### Тональностный профиль
```
12B(E) → 12B(E) → 11B(A) → 6B(A#) → 11B(A) → 11B(A) → 11B(A) → 6B(A#) → 5B(D#) → 12B(E) → 12B(E)
```

**Анализ:** нет строгого гармонического лифта (+1 по Camelot). Есть **блоки**:
- Треки 1-2: E (12B) — начало
- Треки 3,5-7: A (11B) — центральный блок минорного транса
- Трек 4,8: A# (6B) — контрастные вставки
- Трек 9: D# (5B) — самый низкий энергетический уровень
- Треки 10-11: E (12B) — возврат к началу, циклическая структура

**Вывод: энергетическое сведение, а не гармонический лифт.**
Oakenfold выстраивает историю через энергию и контраст, а не через повышение тональности.

### Ключевые выводы по Tranceport

1. **BPM всех треков совпадает — 136.0.** Никакого pitch shift. Все треки изначально в этом темпе.
2. **Breakdown-on-breakdown подтверждён.** Deep minima (3-8%) — ритм почти исчезает, остаётся только мелодия.
3. **Long blends есть, но не на все треки.** Некоторые переходы плотные (15-30 с), некоторые — длинные (2+ мин).
4. **Нет Bass Swap** в явном виде. Энергия растёт постепенно после breakdown, без резкой подмены.
5. **Нет гармонического лифта.** Блочная структура по тональностям, не +1 Camelot.

## Сравнение: Fantazia 1997 vs Tranceport 1998

| Параметр | Fantazia 1997 | Tranceport 1998 |
|----------|---------------|-----------------|
| **BPM** | 129.2 | 136.0 |
| **Жанр** | Хаус | Аплифтинг-транс |
| **Переходы** | 62 (короткие 30-60с) | ~24 (средние, с breakdown) |
| **Треков** | 13 за 72 мин | 11 за 73 мин |
| **Breakdown** | Нет | Да, глубокие (3-8%) |
| **Bass Swap** | Нет | Нет (вопреки гипотезе) |
| **Гармонический лифт** | Нет | Блочный, не линейный |
| **Long blends** | Нет | Частично |
| **Стиль** | Плотный хаус, гладкое сведение | Драматический транс, эпизоды |

### Эволюция техники
1994 (The Goa Mix) → 1997 (Fantazia) → 1998 (Tranceport)

- **1997:** Диджей-сет для танцпола. Единый темп, плотные короткие переходы, без эффектов.
- **1998:** Студийный альбом-микс. Драматическая структура, breakdown, долгие наложения, контрастные блоки.

**Гипотеза из книг (DJ Skills) о Bass Swap и +1 Camelot не подтверждена на Tranceport.** Техника Оукенфолда — не про точные EQ-расчёты, а про **энергетическую архитектуру** и выбор треков, которые «рассказывают историю».

## Раздел 3: Данные из вторичных источников (⚠️ устаревшая гипотеза)

*Ниже — компиляция из книг, статей и интервью. Частично опровергнута анализом Tranceport.*

### 1. Длинные «кинематографичные» переходы (Long Blends)
- Ожидание: 1.5–3 минуты
- Реальность на Tranceport: **частично** — есть 2-минутные, есть 15-секундные

### 2. Паттерн эквализации: Bass Swap
- Ожидание: Bass = -∞ → Bass Swap на 64 такте
- Реальность на Tranceport: **❌ Не подтверждено.** Энергия растёт плавно.

### 3. Сведение «Яма в Яму» (Breakdown Mixing)
- Ожидание: кульминационные паузы накладываются
- Реальность на Tranceport: **✅ Подтверждено** — минимумы 3.5%, 4.3%, 6.3%

### 4. Гармонический лифт
- Ожидание: +1 по Camelot
- Реальность на Tranceport: **❌ Блочная структура**, не линейный лифт

### Источники
- DJ Skills book, The Goa Mix (1994), Perfecto Fluoro (1996)
- Статьи: «DJ School: Разные способы сведения»
