# Paul Oakenfold — Техника сведения

Источник: анализ сетов 1994-2000, интервью, DJ Skills book

## 1. Длинные «кинематографичные» переходы (Long Blends)
- Стандартный переход: **1.5–3 минуты** (64–128 квадратов)
- Входящий трек запускается на самом тихом участке (интро без бочки)
- Накладывается поверх плотного финала уходящего трека
- Слушатель не замечает границы между треками

## 2. Паттерн эквализации: работа с Bass-Line
- **Фаза 1 (Вход):** Bass = -∞, Mid/High на 9–10 часах
- **Фаза 2 (Слияние):** Mid/High плавно до 12 часов, мелодии переплетаются
- **Фаза 3 (Подмена баса):** На 32-м или 64-м такте — быстрая подмена: Low старого → 9 часов, Bass нового → 12 часов. Даёт мощный энергетический толчок

## 3. Сведение «Яма в Яму» (Breakdown Mixing)
- Фирменный приём со времён The Goa Mix (1994)
- Находит треки с совпадающими по тональности кульминационными паузами (Breakdowns)
- «Яма» входящего трека накладывается на «яму» уходящего
- Ритм-секция полностью исчезает, переплетаются только эмбиент-пады, вокал, синты
- Максимальный драматический эффект

## 4. Гармоническое сведение + драматическая структура
- Один из первых топ-диджеев, использующих Camelot Wheel
- **Энергетический лифт:** каждый следующий трек на ступень выше по тональности (например, Am → Em). Создаёт подсознательное ощущение подъёма
- **Кино-нарратив:** Интро (эмбиент/брейкс) → Завязка (прогрессив-хаус) → Кульминация (быстрый гоа/аплифтинг-транс) → Эпилог (глубокий эмоциональный финал)

## 5. FX и сэмплы
- Вставки цитат из научно-фантастических фильмов («Бегущий по лезвию», «Дюна»)
- Космические звуковые эффекты через Pioneer EFX-500 (сирены, реверберированные шумы)
- Маскировка стыков треков с разным темпом/тональностью

## Параметры для пресета (консолидированные)
- **Crossfade:** 1.5–3 min (очень длинный, по сути layered mixing)
- **Tempo:** lock tight ±0.5 (все треки синхронизированы)
- **EQ:** 
  - Bass: полный cut на входе, быстрая подмена на 32/64 такте
  - Mid: плавное выведение
  - High: аккуратное, без резкости
- **Harmonic:** Camelot Wheel, ступенчатый подъём (+1 вверх)
- **Structure:** breakdown-on-breakdown (яма на яму)
- **Curve:** exponential (медленное нарастание, быстрая подмена баса)
- **FX:** внешние сэмплы/фильтры для маскировки стыков
- **Source:** The Goa Mix (1994), Perfecto Fluoro (1996), DJ Skills book
